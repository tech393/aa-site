## STEP 1 — TRIAGE THE EXISTING 266 (90 MIN)

The starting pile is at `/sessions/.../mnt/outputs/michael-faves/` (or the original `Archive.zip` in `/uploads/`). 266 images Michael curated himself. Roughly 78% pass the 5-question filter (validated 2026-05-26).

**Sub-steps:**

1. Extract the full `Archive.zip` into `/PHOTO-CANON-APPROVED/_inbox/` as a working folder.
2. Open each image in Finder Preview (Cover Flow or grid view).
3. For each image, run the **5-question YAY/NAY filter** from PHOTO-STYLE-GUIDE.md:
   - Light (golden hour / sunset / sunrise / warm)?
   - Palette (gold/orange/pink/purple/earth)?
   - Subject (vast nature OR person NOT facing camera)?
   - Framing (negative space, room to project in)?
   - Vibe (contemplative, sacred, inviting)?
4. Move each image into the right `PHOTO-CANON-APPROVED/0X-[category]/` subfolder.
5. NAYs go to `_archived-not-canon/`. Don't delete — keep them in case Michael wants a second look.
6. When in doubt, send to Claude with the guide attached and ask for a YAY/NAY/Borderline call. (Don't ask Michael yet — that's Step 5.)

**Auto-NAY shortcuts** (no need to run the full filter):
- Eye contact with camera → `_archived-not-canon/`
- Office / desk / meeting room → `_archived-not-canon/`
- Posed coaching stock (laptop on couch, headset, blazers) → `_archived-not-canon/`
- White-background headshots → `_archived-not-canon/`
- Cold-light or midday-sun shots → `_archived-not-canon/`

**Expected output after Step 1:** ~200 images sorted into the 11 category folders. ~60 in `_archived-not-canon/`.
