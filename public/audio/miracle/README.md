# Miracle Journal audio files

Drop the 9 mp3s into this folder using the exact filenames below. The Miracle
page (`app/miracle/MiracleClient.tsx`) references these as `/audio/miracle/…`.

If you rename a file, also update its `src:` in `MiracleClient.tsx`.

## Required filenames

1. `01-1-Miracle-Journal-Explanation.mp3`
2. `02-2-Creating-Your-Sacred-Intentions.mp3`
3. `3-Sacred-Appreciation-Feel-The-Gratitude.mp3`
4. `04-4-Yes-It-Is-Done.mp3`
5. `5-Your-Sacred-Rituals-Inspired-Actions.mp3`
6. `7-Your-Weekly-Reflections.mp3`
7. `8-How-To-Make-Sure-You-Do-It.mp3`
8. `Missing-Manifestation-Secret-Simple-but-essential.mp3`
9. `Advanced-Law-Of-Attraction-Training.mp3` (the bonus track)

The numbering quirks (e.g. track 6 = file `7-…`, track 1 = file `01-1-…`)
are intentional — they match the original WordPress upload filenames so the
mp3s drop in without renaming.
